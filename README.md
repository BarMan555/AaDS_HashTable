# AaDS_HashTable
Вторая лабораторная работа ученика Шестерикова Бориса 6213 по АИСД. Хэш-таблицы

Вариант: 24

Mетод открытой адресации
Метод деления: h(k) = k mod m
C помощью хэш-функции, используюшей алгоритм Пирсона, 
получите хэш введенной пользователем строки, содержащей англоязычный текст и сохраните его. 
Напишите функцию, сравнивающую сохраненный хэш с хэшем другой поданной строки. 
Функция должна возвращать true или false.
